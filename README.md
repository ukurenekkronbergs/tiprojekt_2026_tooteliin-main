# tiprojekt_2026_tooteliin
Tooteliini kvaliteedikontrolli projekt.


## Keskkonna seadistamine

Järgige neid samme, et seadistada Pythoni keskkond. See projekt kasutab sõltuvuste haldamiseks condat ja ühilduvuse tagamiseks Python 3.11 versiooni.

1. Lisage oma projekti kausta `environment.yml` fail.
2. Avage terminal (Anaconda Prompt või PowerShell vms, saate teha seda nt VS Code'i terminalis). Vaadake, et olete kaustas, kus .yml fail asub.
3. Loo uus keskkond käsuga `conda env create -f environment.yml`.
4. Aktiveeri keskkond käsuga `conda activate noo`.
